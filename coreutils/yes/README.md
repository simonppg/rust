# yes

yes command print `y` repeatedly

## Run
```
cargo run                 # Repeatedly output 'y'
cargo run -- -h           # Shows help
cargo run -- --help       # Shows help
cargo run -- -v           # Shows version
cargo run -- --version    # Shows version
cargo run -- str1 str2    # Repeatedly output str1 str2
cargo run -- "str1 str2"  # Repeatedly output str1 str2
```
